Textbooks in econometrics and machine learning¶

This list target free and open textbooks (as a website or legal free PDF). Earlier and foundational books also mentioned.

Causal inference¶

Open:

  • The Effect [HK21]

  • The Mixtape [Cun21]

  • Causal Inference for The Brave and True

Earlier:

  • The Book of Why [PM18]

  • Mostly Harmless Econometrics [AP09]

  • Causal Inference for Statistics, Social, and Biomedical Sciences [IR15]

Bayesian statistics¶

Open:

  • Bayesian Data Analysis [Gel14]

    • course

  • Bayesian Modeling and Computation in Python [Mar22]

  • Think Bayes [Dow21]

Earlier:

  • Statistical Rethinking [McE20]

    • video lectures

  • Bayesian Statistics the Fun Way[Kur19]

Econometrics¶

Open:

  • Regression and Other Stories [GHV21]: website, PDF, code repo and predecessor book

Earlier:

  • Peter Kennedy [Ken08] (very appealing to intuition)

  • Stock and Watson [SW11] (expensive when buying new)

Foundational:

  • Hamilton on time series [Ham94]

Probability¶

Open:

  • Probability book [BH15]: PDF, stat101 videos

  • Modern intro to probability and statistics [Dek05]: PDF

Earlier:

  • Probability Theory: The Logic of Science by E.T. Jaynes. [JB03]: first three chapters, reading of the book by Aubrey Clayton.

Machine learning¶

Open:

  • Scipy Lecture Notes

  • ISLR [JWHT21] (now in second edition)

Earlier:

  • Approaching (Almost) Any Machine Learning Problem (inexpensive, no code repo)

  • Hands-on Machine Learning [Geron19]

Papers:

  • Machine Learning Methods That Economists Should Know About [AI19] (also as a working paper)

  • Big Data: New Tricks for Econometrics [Var14]

Talks:

  • P Huenermund on “Causal Inference in Machine Learning and AI (with lessons for econometrics)” (lecture and slides)

Data analysis¶

  • Data Science at the Command Line

  • Julia for Pythonistas.

BH15

Dow21
